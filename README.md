使用 tp5.1 集成一些常用功能
前端使用 bulma, vue 来做
使用 restful 的 api 风格
后端数据库使用 mysql，orm 来做关联查询

功能：
- [ ] jwt
- [ ] oauth
- [ ] 权限控制
- [ ] 缓存（文件缓存 & redis）
- [ ] 验证码
- [ ] 单元测试
- [ ] 数据库迁移
- [ ] 数据填充 faker

使用的composer包

composer require topthink/think-migration=2.0.*

composer require --dev symfony/var-dumper

composer require firebase/php-jwt
